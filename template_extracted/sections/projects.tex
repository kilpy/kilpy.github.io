\subsection{{Deep Learning: Computational Fluid Dynamics Surrogate}}
\subtext {Python: PyTorch, MLP Baseline, Transformers, MLA Attention, Positional Encoding}
\begin{zitemize}
\item Developed a physics-informed transformer-based surrogate model to accelerate computationally expensive CFD simulations on turbulent fluid flow prediction, integrating the Navier-Stokes equations into the loss function to enhance outputs.
\item Using encoded sequential flow field frames from the Cylinder Flow dataset, structured encoder-only and decoder-only transformer models for next frame regression and multi-frame regression respectively.
\item Implemented a custom positional encoding method, ReyPR, infusing the fluid’s Reynold’s Number to the model. Proven next frame regression via PE ablation study. 
\end{zitemize}
